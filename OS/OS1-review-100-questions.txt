TYPE: fill
ANSWER: system call

QUESTION: Explain, in your own words, why a user program cannot simply access a keyboard or screen directly and must go through the operating system instead.
TYPE: short
EXPLAIN: Direct hardware access would require every program to know low-level, device-specific details, would let programs interfere with each other and the hardware, and would tie programs to one specific device. The OS instead provides a simple, safe, and portable abstraction (a system call interface) that hides those details.
